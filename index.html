Graphing

Probability Distributions

All lines will be displayed in the top left corner along with the various options to edit the line.

To select a line, press on the number in the first column of the line you wish to select, This will cause the button to have a background of white and will therefore be selected (line 1 is selected by default).

To add something to the line, use the buttons in the bottom left of the screen. You can cycle between different types of inputs by pressing the buttons above the table of input buttons.

To delete an input, select the desired line and then press the back arrow button.

To say whether you want that line to be graphed make sure that the row with the line has a YES in the 4th column. If it says NO, click it to change it to YES.

The G represents whether you'd like the gradient function to be graphed, click it to change the current state.

To delete a line, press the delete button in the row of the line you want to delete

Once you have set all the parameters to your choice, press the calculate button to graph the equation

The Other tables on the screen will ask for number inputs, if you wish to change which line the table will operate on then you need to change the currently selected line by pressing on the line number in the top left table

you will know which line is selected by the colour of the background of the number being white and the fact that the line number will be shown in the table you are using.

The special points table shows all intersections with the x and y axis aswell as all stationery points

To zoom in or out, hover over the graph and use the mouse wheel, the X domain is shown in the middle on the left.
